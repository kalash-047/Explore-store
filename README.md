## Features

- **8 Micro-markets analyzed:** Indiranagar to Electronic City
- **Pincode-level scoring:** 0-100 viability score
- **Format recommendation:** Flagship vs Community vs Partnership
- **Financial projections:** Investment, break-even, revenue
- **Cannibalization risk:** Online vs offline conflict analysis
